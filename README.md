# 🔥 火炎之紋章：翠星之影

> Fire Emblem: Shadow of the Emerald Star

一款完全在瀏覽器中運行的戰略角色扮演遊戲（SRPG），靈感來自經典的《火焰之紋章》系列。

## 📖 故事

帝國宰相莫爾甘發動政變，國王犧牲自己讓公主艾琳逃出王宮。艾琳帶著翠星之印踏上復國之路，途中結識了忠誠的騎士馬庫斯、弓手莉娜、戰士托爾、修女賽拉、傭兵凱恩、天才魔法師法蘭等夥伴。穿越荒野、森林、要塞，最終攻入帝都，擊敗莫爾甘，奪回和平。

全 11 章（序章 + 10 章）的完整劇情，包含對話、招募、轉職系統。

## 🎮 遊玩方式

直接用瀏覽器開啟 `index.html`，或訪問 GitHub Pages 部署版本。

### 操作

| 操作 | 滑鼠 | 鍵盤 |
|------|------|------|
| 移動游標 | 滑鼠移動 | WASD / 方向鍵 |
| 選擇/確認 | 左鍵點擊 | Enter / Z |
| 取消 | — | Esc / X |
| 查看狀態 | — | R |
| 結束回合 | 點擊按鈕 | — |

### 基本玩法

1. **選擇單位** → 出現指令選單
2. **移動** → 藍色範圍內點選目標格
3. **攻擊** → 選擇敵人，查看戰鬥預測後確認
4. **待機** → 結束該單位行動
5. **結束回合** → 進入敵軍回合

### 系統特色

- ⚔️ **武器三角**：劍 > 斧 > 槍 > 劍 / 火 > 風 > 雷 > 火
- 🏔️ **地形效果**：森林、山地、城門等提供防禦和迴避加成
- 📈 **成長系統**：升級時根據成長率隨機提升屬性
- 🗣️ **對話招募**：特定角色可以透過對話加入隊伍
- 💾 **自動存檔**：每章結束自動存檔至瀏覽器 localStorage

## 🛠️ 技術

- 純 HTML5 Canvas + JavaScript，無框架依賴
- 所有精靈圖為程式化像素藝術（procedural pixel art）
- Web Audio API 音效（振盪器合成，無外部音檔）
- 角色肖像為 AI 生成 PNG
- 100% 前端，可部署至任何靜態託管服務
