## Enemy Stat Formula

Generated by `game.js:generateEnemyStats(classId, level)`:

```
hp  = 16 + level × 2
str = 3 + level        (physical classes)
mag = 3 + level        (magic classes — those with fire/thunder/wind/dark/staff)
skl = 2 + level
spd = 2 + level
lck = 1 + floor(level / 2)
def = 2 + level
res = 1 + floor(level / 2)
```

`bonusStats` in config adds on top of these. Any enemy can have `bonusStats`, not just bosses.

## Difficulty Tuning (as of 2026-05)

Chapters were reviewed for difficulty with a preference for **hard**. The main changes:

- ch3: expanded from 0 → 15 enemies
- ch6: expanded from 4 → 21 enemies
- ch7–ch10: all filled from stub (2–3 enemies) to 23–32 enemies with reinforcement waves

Later chapters (ch7+) use promoted-class enemies: `warrior`, `sniper`, `paladin`, `mageKnight`, `general`, `swordmaster`, `sage`. These are stronger than the base-class enemies in earlier chapters.

Boss bonusStats scale up chapter by chapter:
- ch7 boss: fenrir + nosferatu, HP ~52, MAG ~21
- ch8 boss: silverLance + silverSword, HP ~40, STR ~14, DEF ~15
- ch9 boss: fenrir + nosferatu on throne tile (heals), HP ~50, MAG ~21
- ch10 boss: starCrest + fenrir, HP ~60, MAG ~25 — the starCrest (might:20, hit:90, crit:10) makes this the most dangerous enemy in the game
